# TrailSnap CLI

Command Line Interface for TrailSnap.

## Installation

### Via pip

```bash
pip install trailsnap-cli
```

### Via npm

```bash
npm install -g trailsnap-cli
```

## Usage

```bash
trailsnap --help
```

获取TrailSnap的API URL和Token

```bash
trailsnap config set --url <url> --token <token>
```
