## api响应改造

package\server\app\api 里的所有接口都统一改成返回json格式
返回格式：
```json
{
    "code": 0,
    "msg": "success",
    "data": {}
}
```
其中，code表示状态码，msg为状态信息，data为返回数据
如果失败，msg为失败信息

**只修改后端api**，不要修改前端代码

参考package\server\app\api\ocr.py

移动图片到指定目录功能

1. package\website\src\components\PhotoLightbox.vue 里新增一个按钮，点击后弹出一个弹窗，用户可以在弹窗里选择要移动的目录（可以新建目录，目录必须是用户添加的外部文件夹，不能是没权限的其他目录）
点击确认后，调用后端api移动图片到指定目录，并更新图片元数据
2. 相册新增移动或者复制图片到某个文件夹的功能，文件夹名默认是相册名，用户可以在弹窗里修改
3. 人物相册、位置相册、智能分类均新增移动或者复制图片到某个文件夹的功能，文件夹名默认是相册名，用户可以在弹窗里修改
4. 工具箱新增按时间整理图片的功能，用户可以在弹窗里选择要整理的时间范围，点击确认后，调用后端api按时间整理图片，将图片按年月分类整理到对应的文件夹里

工具箱（package\website\src\views\toolbox\ToolboxPage.vue）新增图片文件整理功能，支持按照时间（年月，年月日）把图片整理到对应的文件夹里，按照分类把图片整理到对应的文件夹里，按照人物把图片整理到对应的文件夹里，这是一个工具页面

再新增一个选项，按位置把图片整理到对应的文件夹里
支持省、市、区三级可选，可以平铺结构也可以递归结构

工具箱（package\website\src\views\toolbox\ToolboxPage.vue）新增批量重命名功能

1. 新增一个文件重命名页面，用户可以选择指定文件夹里的图片，按照一定的规则重命名图片文件
2. 支持按时间（YYYYMMDD_HHMMSS）重命名(数据库里的photo.photo_time)
3. 支持添加前缀或后缀，默认前缀：IMG_
4. 对于相同时间的图片，按顺序添加后缀，后缀从(1)开始递增
5. 文件后缀名与原来保持一致

---

工具箱（package\website\src\views\toolbox\ToolboxPage.vue）新增根据文件名修改图片拍摄时间功能

1. 新增一个页面，用户可以选择指定文件夹里的图片
2. 支持按时间（YYYYMMDD_HHMMSS）(数据库里的photo.photo_time)修改图片拍摄时间（文件元数据）
3. 只修改支持exif元数据的图片格式，对于不支持的图片格式，修改系统属性，而不是文件元数据
4. 可选项：仅修改元数据缺失的图片，拍摄设备信息（品牌、型号）

不需要使用现有工具提取出文件名中的时间信息，直接用数据库的photo.photo_time字段就行了，也不需要更新数据库的photo_time数据，因为photo.photo_time字段本身就是根据文件名和元数据获取的

照片拍摄时间设置为可选项
- 自动识别（默认，按照现有的处理逻辑）
- 指定时间（用户可以在弹窗里指定时间，所有照片都修改为该时间）

后台任务优化

package\server\app\core\config_manager.py 新增配置项，用于设置内置的ai大模型连接，模型地址为 ai_api_url/v1（ai_api_url可能会修改）,api_key为"empty"，大模型智能分析和ai对话默认配置为内置的连接

如果用的是默认的ai大模型，则package\server\app\service\task_worker.py里的category == 'AI'的任务不再使用单独的任务队列，而是与category == 'IO'的任务共享一个队列，避免同时处理消耗过多的资源
