# 设置指南

TrailSnap 提供了丰富的自定义选项，帮助您打造个性化的智能相册。

## 主要配置项

- **基础设置**：外观主题、存储路径、AI 参数配置等。
- **地图设置**：[地图 API Key 配置](./mapsetting.md)，用于开启足迹地图功能。
- **Token 设置**：[访问令牌配置](./tokensetting.md)，用于第三方应用访问后端 API。
- **任务管理**：查看后台任务状态（索引重建、人脸识别等）。

## 配置文件

除了通过 Web 界面修改设置外，高级用户也可以直接修改配置文件（通常位于数据目录下的 `config.json`）。但为了避免格式错误，建议优先使用 Web 界面进行操作。
