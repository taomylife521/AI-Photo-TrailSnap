# 功能清单

本文档列出 TrailSnap（行影集）的完整功能列表，帮助你快速了解应用的所有能力。

---

## 1. 相册管理

### 1.1 基础功能

| 功能 | 说明 |
| --- | --- |
| 照片上传 | 支持多文件上传，自动生成缩略图 |
| 照片浏览 | 瀑布流/网格视图，支持筛选和过滤 |
| 照片灯箱 | 全屏照片查看器，带导航功能 |
| 照片元数据 | EXIF 数据提取与展示 |
| Live Photo | 支持 iPhone、Vivo、OPPO、小米等手机的实况照片 |
| 回收站 | 软删除功能，支持恢复已删除的照片 |

### 1.2 智能分类

| 功能 | 说明 |
| --- | --- |
| 人脸识别 | 基于 InsightFace 的人物识别与聚类 |
| 场景分类 | 自动分类为夜景、宠物、美食、自拍等场景 |
| OCR 识别 | 从照片中提取文字信息 |
| 相似照片检测 | 查找视觉上相似的照片 |
| 重复照片清理 | 基于哈希值的重复照片检测与清理 |
| AI 照片分析 | AI 生成照片描述和评分 |

---

## 2. AI 能力

| 功能 | 说明 |
| --- | --- |
| AI 助手对话 | 与 AI 大模型对话，智能检索相册内容 |
| 智能相册生成 | 根据描述自动生成相册（如"我在海边的自拍"） |
| 那年今日 | AI 推荐往年今天最具回忆价值的照片 |
| 智能搜索 | 支持按人物、内容、时间、地点等维度搜索 |
| 多模型支持 | 支持连接多个 AI 大模型（OpenAI、Ollama 等） |

---

## 3. 足迹与地图

| 功能 | 说明 |
| --- | --- |
| 足迹地图 | 在交互式地图上查看所有照片 |
| 点亮的城市 | 统计并展示去过的城市，支持省/市/区县级别 |
| 去过的景区 | 追踪去过的 5A 级景区，支持添加自定义景区 |
| 足迹时间轴 | 按时间轴展示不同地点的照片 |
| 轨迹视图 | 将一段时间内的地点连接起来，展示移动轨迹 |
| 离线地图 | 支持下载多国离线地图数据 |

---

## 4. 行程票据

### 4.1 火车票

| 功能 | 说明 |
| --- | --- |
| 火车票识别 | OCR 自动识别照片中的火车票信息 |
| 火车票管理 | 增删改查火车票记录 |
| 纸质票样式 | 仿真火车票展示（红票/蓝票样式） |
| 火车票导出 | 支持导出为 PNG、CSV、JSON 格式 |

### 4.2 飞机票

| 功能 | 说明 |
| --- | --- |
| 飞机票识别 | 识别照片中的行程单截图 |
| 飞机票管理 | 增删改查飞机票记录 |
| 机票统计 | 按城市、距离、时长等维度统计 |

### 4.3 票据统计

| 功能 | 说明 |
| --- | --- |
| 年度票据报告 | 出行统计摘要 |
| 城市统计 | 去过的城市及照片数量 |
| 距离统计 | 旅行总里程 |
| 乘客筛选 | 按乘客筛选票据记录 |

---

## 5. 相册类型

| 相册类型 | 说明 |
| --- | --- |
| 普通相册 | 传统相册功能，按时间线组织 |
| 智能相册 | AI 根据描述自动生成 |
| 人物相册 | 人脸识别后的人物照片集合 |
| 位置相册 | 按地理位置组织的照片 |
| 场景分类相册 | 按场景类型（夜景、美食等）分类 |
| 条件相册 | 支持按标签、场景、人物、地点等条件筛选生成 |

---

## 6. 年度报告

| 功能 | 说明 |
| --- | --- |
| 照片墙 | 年度照片拼图展示 |
| 照片日历 | 按日期展示年度照片 |
| 城市榜单 | 去过的城市排名 |
| 景区榜单 | 去过的景点排名 |
| 行程时间轴 | 年度行程时间线 |
| 线路里程 | 旅行线路与总里程 |
| 季节分析 | 春夏秋冬各季节照片统计 |
| 消费统计 | 机票、火车票消费统计 |
| 最佳照片 | 年度 AI 评分最高的照片 |
