# Settings Guide

TrailSnap provides rich customization options to help you build a personalized intelligent photo album.

## Main Configurations

- **Basic Settings**: Appearance theme, storage path, AI parameter configuration, etc.
- **Map Settings**: [Map API Key Configuration](./mapsetting.md), used to enable the footprint map function.
- **Token Settings**: [API access token](./tokensetting.md), used for third-party apps to access backend APIs.
- **Task Management**: View background task status (index rebuilding, face recognition, etc.).

## Configuration Files

In addition to modifying settings via the Web interface, advanced users can also directly modify configuration files (usually located in `config.json` under the data directory). However, to avoid format errors, it is recommended to prioritize using the Web interface for operations.
